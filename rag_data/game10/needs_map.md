# Карта потребностей → рекомендация

## Деньги / карьера
Триггеры:
- “нет финансового плана”, “потолок”, “работаю за троих”, “не ценят”, “страшно повышать цену”
Рекомендация:
- «Игра 10:0» (неделя “Деньги”) + “Финансовая распаковка Рода” как старт

## Отношения
Триггеры:
- “притягиваю холодных”, “спасаю”, “вина”, “терплю”, “нет границ”
Рекомендация:
- «Игра 10:0» (неделя “Отношения”) + чек-лист “Отношения 10:0”

## Энергия / ресурс
Триггеры:
- “выгорание”, “нет сил”, “апатия”, “устал(а)”
Рекомендация:
- «Игра 10:0» (неделя “Энергия”) + ресурсная среда вместо самополомки

## Таланты / проявленность
Триггеры:
- “я недостойна”, “ещё рано”, “страшно проявляться”, “я не знаю чего хочу”
Рекомендация:
- «Игра 10:0» (неделя “Таланты”) + аудио “Сам себе Фанат”

## Как ассистент должен формулировать предложения
- 1–2 предложения эмпатии
- 1–3 варианта (если уместно: курс + консультация + мероприятие)
- следующий шаг: “написать ИГРА” или “связаться с менеджером”
